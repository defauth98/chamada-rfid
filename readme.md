<h1 align="center">Chamada RFID</h1>

### :thinking: Como rodar

```bash
# Clone o repositório
git clone https://github.com/defauth98/chamada-rfid.git chamada-rfid

# Entre no diretório
cd chamada-rfid

# Instale as depedências
npm install

# Rode o projeto
npm start
```
